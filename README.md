# TrashCash

Scan your trashbarcodes get free money.

## Getting Started

Super easy. Install the requirements, then just python trashCash.py and you ready to rock. For rapi's, just get a picam and edit out the appropriate lines.

### Prerequisites

Listed in code.
  - PIL
  - picamera (for picam)
  - openCV
  - pyota
